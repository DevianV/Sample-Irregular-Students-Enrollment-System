# Images Directory

Place your images here:

## Required Images

1. **PLM Logo**
   - File name: `plm-logo.png` (or `.jpg`, `.svg`)
   - Recommended size: 200x200px or larger
   - Format: PNG with transparency (preferred) or JPG

2. **Login Background**
   - File name: `login-bg.jpg` (or `.png`)
   - Recommended size: 1920x1080px or larger
   - Format: JPG (for smaller file size) or PNG

## File Structure

```
/images
  ├── plm-logo.png      (PLM logo)
  ├── login-bg.jpg      (Login page background)
  └── README.md         (This file)
```

## Alternative File Names

If you use different file names, update `login.php`:
- Logo: Change `plm-logo.png` to your filename
- Background: Change `login-bg.jpg` to your filename

## Image Optimization Tips

- **Logo**: Use PNG with transparency for best quality
- **Background**: Use JPG for smaller file size (recommended: < 500KB)
- Compress images before uploading for faster page load
